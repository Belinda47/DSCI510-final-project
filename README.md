# DSCI 510 Final Project

This project analyzes social media engagement across TikTok, Instagram, and YouTube.

The goal is to explore factors that influence engagement metrics such as views, likes, comments, and shares.

Data sources include:
- TikTok dataset from Kaggle
- Instagram dataset from Kaggle
- YouTube Data API

The project is implemented in Python using libraries such as pandas and google-api-python-client.
